# 🌸 Period Tracker

A modern PHP & MySQL web application for menstrual cycle tracking.

## Features

- 🔐 User Registration & Login
- 📅 Calendar View
- 🌸 Period Tracking
- 🔮 Predicted Period Days
- 📖 Personal Diary
- 📊 Reports & Statistics
- 👤 Profile Management
- ⏳ Automatic Session Timeout

## Technologies

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
